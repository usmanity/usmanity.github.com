Dashes

Use this page to copy different types of dashes.
Clicking anywhere on the dash will copy it.
Holding Shift while clicking will copy the Unicode point.
Holding ALT while clicking will copy the HTML reference.
Dash font-size is 36px, dash name and discription are 16px.

  • —
    Em dash
    Unicode point U+2014 HTML reference &mdash;
  • –
    En dash
    Unicode point U+2013 HTML reference &ndash;
  • ―
    Horizontal bar
    Unicode point U+2015 HTML reference &horbar;
  • -
    Hyphen-minus
    Unicode point U+002D HTML reference &hyphen;
  • ⁓
    Swung dash
    Unicode point U+2053 HTML reference &8275;
  • ‒
    Figure dash
    Unicode point U+2012 HTML reference &8210;
